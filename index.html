    Bounty Bar: API REST

    Sistema de Gestión de clientes y promociones en bar/discoteca. Documentacion tecnica completa de todos los endpoints del backend. Cada endpoint incluye metodo HTTP, ruta, nivel de autorizacion, parametros, cuerpo de solicitud y respuesta esperada. El sistema incluye tareas automaticas: auto-cancelacion de pedidos pendientes (6 min) y reinicio mensual de puntos actuales (dia 1 de cada mes).

    Autenticacion

    Los endpoints protegidos requieren un token JWT en el header Authorization: Bearer <token>. El token se obtiene al iniciar sesion en POST /auth/login. Los endpoints marcados como Admin requieren ademas que el usuario tenga rol de administrador.
